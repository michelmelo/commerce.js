# Security policy

## Supported versions

Commerce.js versions 1 and 2 are currently supported, although we recommend upgrading to 2.x if you are using 1.x.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| 2.x     | :white_check_mark: |
